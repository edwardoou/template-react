# Start

```zsh
npm i
npm run dev
```

## Notes
- Eliminar el archivo *.gitkeep* de las carpetas "assets, components, layouts y utils", este archivo es una marca de tiempo que se utiliza para que el sistema de versionamiento funcione correctamente.
